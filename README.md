# Cab Hiring App (Web)

This folder contains the web-only pivot of the Sprint-1 app, implemented in Next.js.

## Run locally

From the repository root:

1. Install web dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

Then open http://localhost:3000

## Demo login

- Enter any user ID + any password
- OTP is **123456**

## Notes

- Data is persisted in `localStorage`.
- This is a functional demo (no real payments, no real OTP, no real KYC backend).
